# mod-devel-cli

## Installation

Run:

    pip install git+https://github.com/mod-devel-cli.git
    
## Usage

```
Usage: modcli auth [OPTIONS] COMMAND [ARGS]...

Options:
  -h, --help  Show this message and exit.

Commands:
  active_token  Show current active access JWT token
  login         Authenticate user
```
